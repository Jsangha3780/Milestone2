# Campus Event Organizer API Routes

## Event Routes

GET /events
Retrieve all campus events.

POST /events
Create a new event.

PUT /events/:id
Update event information.

DELETE /events/:id
Delete an event.

---

## User Routes

GET /users
Retrieve all users.

POST /users
Create a new user.

---

## Registration Routes

GET /registrations
Retrieve all registrations.

POST /registrations
Register a student for an event.

---

## Juice Report Route

GET /juicereport/:id_juice
Generate event statistics and reports.
